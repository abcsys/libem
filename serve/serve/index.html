Include the access token as part of the authorization request header:
Authorization: Bearer <ACCESS TOKEN>
Python example:
requests.get("https://serve.arena.org/credits",
  headers={
    "Authorization": "Bearer <ACCESS TOKEN>"
  }
)
Curl example:
curl https://serve.arena.org/credits -H "Authorization: Bearer <ACCESS TOKEN>"
